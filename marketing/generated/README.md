# Visuels gratuits Comparia

Pack prêt pour posts organiques :

- `pinterest-checklist-factures.svg`
- `pinterest-box-mobile.svg`
- `pinterest-assurance.svg`
- `short-thumbnail-trop-cher.svg`
- `short-thumbnail-300-euros.svg`

Les fonds PNG générés sont dans `backgrounds/`. Les SVG gardent le texte net, modifiable et cohérent avec le logo Comparia.

Formats :

- Pinterest : 1000 x 1500
- Shorts/Reels/TikTok : 1080 x 1920

Utilisation recommandée :

- Publier 1 visuel Pinterest par jour avec le lien du guide correspondant.
- Utiliser les miniatures Shorts sur les vidéos du fichier `marketing/short-video-scripts.csv`.
- Garder les mêmes CTA : `Voir mes économies`, `Comparer gratuitement`, `Faire le diagnostic`.
